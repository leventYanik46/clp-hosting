# Comment les professionnels de sante peuvent travailler aux Etats-Unis

Les Etats-Unis continuent d'attirer des medecins, infirmiers, pharmaciens,
therapeutes et autres professionnels de sante formes a l'etranger. Cependant,
exercer legalement dans le pays exige plus qu'une simple offre d'emploi. Il
faut souvent combiner **reconnaissance academique**, **licence
professionnelle** et **strategie migratoire adaptee**.

## 1. Verification des diplomes et equivalences

Dans de nombreuses professions de sante, les diplomes etrangers doivent etre
evalues avant toute demande de licence. Selon la profession, cela peut inclure
des organismes d'equivalence, des examens nationaux et parfois des formations
complementaires.

## 2. Exigences de licence professionnelle

Les licences sont souvent gerees par les Etats. Cela signifie que les regles
peuvent varier pour :

* les infirmiers ;
* les medecins ;
* les dentistes ;
* les pharmaciens ;
* les autres professions de sante reglementees.

Une personne peut etre qualifiee dans son pays d'origine tout en devant remplir
des conditions supplementaires aux Etats-Unis.

## 3. Options de visa frequentes

Selon le poste, l'employeur et le profil du candidat, plusieurs categories
peuvent etre envisagees :

* **H-1B** pour certains postes hautement qualifies ;
* **TN** pour certaines professions et certaines nationalites admissibles ;
* **EB-2** ou autres categories basees sur l'emploi pour les personnes qui
  visent la residence permanente.

## 4. Pourquoi la planification compte

Dans le secteur de la sante, l'ordre des etapes est essentiel. Demander un visa
trop tot, ne pas verifier les conditions de licence, ou accepter un poste sans
strategie claire peut retarder l'installation professionnelle.
