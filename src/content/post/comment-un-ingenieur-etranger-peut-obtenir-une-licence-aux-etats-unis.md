# Comment un ingenieur etranger peut obtenir une licence aux Etats-Unis

L'ingenierie est l'une des professions les plus reglementees aux Etats-Unis.
Pour un ingenieur forme a l'etranger, obtenir une licence professionnelle peut
etre indispensable pour evoluer, acceder a des postes plus qualifies et signer
des documents techniques de facon officielle.

## 1. Comprendre le systeme americain

Les licences sont gerees **au niveau des Etats**. Chaque conseil d'Etat fixe
ses propres exigences, mais le parcours general comprend souvent :

1. l'evaluation des diplomes ;
2. l'examen **FE** ;
3. l'experience professionnelle requise ;
4. l'examen **PE** ;
5. la demande finale d'enregistrement.

## 2. L'evaluation des diplomes

Avant les examens ou le depot d'un dossier, le diplome obtenu hors des
Etats-Unis doit en general etre verifie par un organisme reconnu, comme :

* **NCEES Credentials Evaluation** ;
* **ECE** ;
* **WES**.

Cette etape permet de determiner si la formation est equivalente a un programme
accredite aux Etats-Unis.

## 3. Les examens et l'experience

L'examen **Fundamentals of Engineering (FE)** est souvent la premiere grande
etape. Une fois reussi, de nombreux Etats exigent une periode d'experience
supervisee avant de permettre l'acces au **Professional Engineer Exam (PE)**.

## 4. Immigration et licence doivent avancer ensemble

Le projet de licence ne peut pas etre separe du bon statut migratoire. Selon le
profil, les options les plus frequentes sont :

* **H-1B** pour les postes specialises ;
* **TN** pour certaines nationalites eligibles ;
* **EB-2** ou **NIW** pour les profils qualifies visant la residence
  permanente.
